# Development environment in vagrant with Ansible automation #

** This is an starter Project for setting development environment in vagrant with Ansible automation

Current version of environment is using:
 * Ansible 2.4
 * apache 2.4
 * PHP 7.1
 * Python 2.7
 * Ruby 2.4
 * Sass 3.5.1
 * Mysql 5.6
 * nodejs 6.11
